Font Size

The font-size property allows you to customize the size of text within HTML elements. You can specify the font size in various units such as pixels (px), ems (em), or percentages (%). For example, to make a header text larger, you can set font-size: 30px;

Hereâ€™s an example of styling a header using CSS:

.header {
padding: 60px;
text-align: center;
background-color: #1abc9c;
color: white;
font-size: 30px;
}
Background Color

The background-color property allows you to change the background color of an element. In the header example above, we set the background color to green using background-color: #1abc9c;

You can experiment with different colors or use named colors like red, blue, or yellow. Remember to choose colors that provide good contrast with the text for readability.

background-color: #1abc9c;
Text Alignment

Adjusting the alignment of header text is essential for a clean layout. Use the text-align property to control alignment

These examples demonstrate centering, left alignment, right alignment, and justified text:

text-align: right
Padding

The padding property adds space around the content within an element. In our header, we set a large padding using padding: 60px;. Adjust this value to create the desired spacing.

Padding helps prevent text from touching the edges of the container, providing a visually pleasing layout.

padding: 60px;
Color

The color property defines the text color. By setting color: white; in our header example, we ensure that the text appears in white against the green background

Remember that if you set the color for the body text, all headings within the body will inherit the same color unless specified otherwise

color: white;
